# Async Weather Tracker

A vanilla JavaScript application designed to demonstrate the power of asynchronous programming, the event loop, and persistent local storage.

## Features
- Real-time Weather Data: Fetches data using the Fetch API with async/await.
- Execution Tracking: Console-style output on the UI to visualize the Event Loop (Call Stack vs. Task Queue).
- Persistent History: Saves recent searches to LocalStorage.
- Error Handling: Graceful management of network issues and invalid city names.

## How to Run
1. Clone the files.
2. Replace 'YOUR_API_KEY' in script.js with a valid key from OpenWeatherMap.
3. Open index.html in any modern browser.
